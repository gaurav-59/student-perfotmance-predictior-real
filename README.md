# student-perfotmance-predictior-real
This is a ml project to predict student marks based of study hours and sleep using Linear regression
Student Performance Predictor

This project uses a simple machine learning model to predict a student’s marks based on the number of hours studied and sleep duration.

Overview

The goal of this project is to understand how basic factors like study time and sleep can influence academic performance. It also demonstrates how raw data can be cleaned and used to train a predictive model.

Features

- Handles missing data using mean values
- Trains a Linear Regression model
- Accepts user input for prediction
- Displays results along with a visual graph

Technologies Used

- Python
- Pandas
- Scikit-learn
- Matplotlib

How to Run

1. Clone or download the repository
2. Make sure both "app.py" and "real_student.csv" are in the same folder
3. Run the script:
   python app.py
4. Enter study hours and sleep hours when prompted
5. View the predicted marks and graph output

About

This project was built as part of my initial learning in Machine Learning, focusing on understanding the complete workflow from data cleaning to prediction and visualization.
